Colophon notes

Character

Hand-tuned, lightweight, and close to the material

Stack

Semantic HTML, bespoke CSS, and minimal JavaScript

Workflow

Make-based preview and staged shell deploy

Type

The site is set in Söhne, mostly in light, book, and halbfett cuts, with italics reserved for contrast, emphasis, and a little mischief.

Build

It runs as a deliberately simple static site: semantic HTML, bespoke CSS, and just enough JavaScript to handle theme state, navigation behavior, and a few quality-of-life details.

Preview

Local previews come up through the repo Make targets, which wrap a tiny Python server so the work can stay inspectable in a browser without adding framework overhead.

Deploy

Shipping goes through a staged shell deploy that syncs the managed site files and refreshes cache-busted asset URLs on the way out, keeping the source tree clean.

Style Guide

Foundations, tokens, and recurring interface patterns live in the site style guide.
